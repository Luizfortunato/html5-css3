Exemplo da Tag CODE com um trecho de código em Python:

Faça um program que leia um número inteiro e mostra na tela o seu sucessor e o seu antecessor

        
            print('Inicio do programa!!!\n')

            N = int(input('Digite um número para saber o seu sucessor e também o seu antecessor: '))

            print('\nO número digitado foi {} o seu sucessor é o número {} e o seu antecessor é {} !!!'.format(N, N+1, N-1))

            print('\nFim do programa!!!')

Usamos a Tag PRE pois ela respeita os espaçoes e a identação original do código.

Mais um exemplo em que a identação é fundamental para a interpretação e funcionamento do código:

Crie um programa que leia duas notas de um aluno e calcule sua média, mostrando uma mensagem no final conforme a média atingida:
— Média abaixo de 5,0: REPROVADO
— Média entre 5,0 e 6,9: RECUPERAÇÃO
— Média 7.0 ou superior: APROVADO

        
            N1 = eval(input('Informe a primeira nota: '))
            N2 = eval(input('Informe a segunda nota: '))
            media = (N1 + N2) / 2
            
            print('Com as notas {:.1f} e {:.1f} sua média final é {:.1f}!!!'.format(N1, N2, media))
            if media < 5.0:
                print('\033[1;31mAluno REPROVADO!!!')
            
            elif 5 <= media <= 6.9:
                print('\033[1;33mAluno em RECUPERAÇÃO!!!')
            
            elif media >= 7.0:
                print('\033[1;32mAluno APROVADO!!!'))

Observe que no código acima a identação e as quebras de linha foram respeitadas devido ao envelopamento da Tag CODE dentro de uma Tag PRE.
