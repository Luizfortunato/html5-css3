Adicionando Emojis

Podemos adicionar emojis utilizando parte do seu código hexadecimal. Exemplo:
Código do emoji: U+1F596
Para ser reconhecido pelo HTML devemos informar que iremos utilizar o padrão Hexadecimal para isso usamos &#x
Dessa forma temos: &#x + 1F596; código completo em hexadecimal
Esse código deve sempre vir acompanhado de um ponto e vírgula.

  • Emoji 😀 Seu código U+1F600
  • Emoji 🤩 Seu código U+1F929
  • Emoji 🤪 Seu código U+1F92A
  • Emoji 🤨 Seu código U+1F928
